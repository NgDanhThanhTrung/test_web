Giải pháp DNS Tự động

Cài đặt trực tiếp vào Settings của iPhone. Shadowrocket tự động chạy khi mở Locket, cực kỳ tiết kiệm pin và không cần bật/tắt App thủ công.

Cài DNS ngay qua FB
1. Bật HTTPS Decryption

• Shadowrocket -> Config -> Dấu (i) của file -> HTTPS Decryption -> Bật Decryption.

• Chọn Generate New CA Certificate -> Install Config Profile.

2. Tin cậy chứng chỉ

• Vào Cài đặt máy -> Đã tải về hồ sơ -> Cài đặt.

• Cài đặt chung -> Giới thiệu -> Cài đặt tin cậy chứng nhận -> Gạt bật Shadowrocket.

3. Không hiện Gold?

Hãy kiểm tra Shadowrocket đã Connected chưa, Module đã thêm chưa và đã tin cậy chứng chỉ chưa. Nếu vẫn lỗi, hãy gỡ Locket tải lại.

4. Có bị khoá acc không?

Không. Module chỉ can thiệp hiển thị Premium nội bộ trên máy bạn, không gửi dữ liệu ra ngoài. Bảo mật 100%.
