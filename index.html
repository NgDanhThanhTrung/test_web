1. Bật HTTPS Decryption

• Shadowrocket -> Config -> (i) của file -> HTTPS Decryption -> Bật Decryption.

• Chọn Generate New CA Certificate -> Install Config Profile.

2. Tin cậy chứng chỉ

• Cài đặt máy -> Đã tải về hồ sơ -> Cài đặt.

• Cài đặt chung -> Giới thiệu -> Tin cậy chứng nhận -> Gạt bật Shadowrocket.
